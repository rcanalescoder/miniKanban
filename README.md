# miniKanban

`miniKanban` es una demo web de un tablero Kanban moderno, claro y visual para enseñar cómo organizar tareas con un flujo sencillo de cuatro columnas: Definido, En curso, Bloqueado y Terminado.

## Cómo instalar

```bash
npm install
```

## Cómo arrancar

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.

## Qué hace la demo

- Muestra un tablero Kanban en una única pantalla.
- Añade una franja superior de personas con Pepe, Juan y Sara, y permite crear más.
- Permite crear, editar, eliminar, mover y reordenar tareas.
- Permite asignar cada tarea a una persona del equipo y muestra su foto y área.
- Incluye edición rápida del título y carga rápida de varias tareas.
- Guarda los cambios en `localStorage` para mantener el estado al recargar.
